# Project Overview
8-puzzle game implemented in React Native

## Install Dependencies
1. Install all the packages
```shell
npm install
```

2. Link any needed packages
```shell
react-native link
```

3. Run ios. (This is tested in ios but feel free to run in android as well)
```shell
react-native ios
```
